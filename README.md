# Stability and interpretability of penalized logistic regression models 

This repository contains the data and R code used in the manuscript:

**Stability and interpretability of penalized logistic regression models for breast cancer risk prediction**

## Repository Structure

- data/ – Breast Cancer Wisconsin Dataset
- R/ – R scripts for preprocessing, model fitting, evaluation, and bootstrap stability selection

## Software

R (version 4.4 or later)

Packages

- glmnet
- caret
- pROC
- rms
- ggplot2
- dplyr
- tidyr
